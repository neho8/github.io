Minecraft Servers Using Docker

Date: 12.10.2023 + Video below


A Great Project For Beginners

In this tutorial, we will walk through setting up multiple Minecraft servers using Docker. Docker is a powerful tool that allows us to containerize applications, making it easy to manage and run various server instances simultaneously. I will take you through the process step by step, from installing the Minecraft server image to configuring and running multiple servers on the same machine.


1. Installing Minecraft Server and Docker

Before beginning the server setup, you need to have Docker installed on your system. If you don't have it already, you can install it from Docker's website. Once it is installed, you are ready to proceed.


2. Obtaining the Minecraft Server Image

To set up the Minecraft server, we will use a Docker image called "itzg/minecraft-server." This image is actively maintained, and it's a great choice for running Minecraft servers. Browse to Docker Hub, find the "itzg/Minecraft-server" image, and paste the Docker Pull Command into your terminal.


3. Configuring the Minecraft Server Parameters

When setting up your Minecraft server, we will configure some essential parameters to ensure it functions correctly. Here's a breakdown of the crucial settings:

Docker Run: This is the command to start the Docker container.

docker run -d

We will use the "-d" flag to detach the container from our terminal and run it in the background.

Port Listing: Define the port you want the Minecraft server to run on.

-p 25565:25565

Name: Set a unique name for your Minecraft server container.

--name mc

EULA: Make sure to include "EULA=TRUE" to accept the End User License Agreement.

-e EULA=TRUE

Restart Policy: ensure the server automatically restarts if it goes down.

--restart=always

Directory: Specify the directory for your Minecraft server data. You can use an existing data file if you have one.

-v C:\Users\conta\Downloads\minecraft\data:/data

Image: The Docker image name and tag you copied earlier.

itzg/minecraft-server

Here is the complete command (one line):

docker run -d -p 25565:25565 --name mc -e EULA=TRUE --restart=always -v C:\Users\conta\Downloads\minecraft\data:/data itzg/minecraft-server


4. Creating and Running the Minecraft Server

Now that you have pulled the docker image and configured the Minecraft server parameters, you can run the server using the Docker command. After running the command, Docker will set up the server for you.


5. Testing the Minecraft Server

To verify that your Minecraft server is up and running, open the Minecraft client, go to the multiplayer section, and enter the IP address of your Docker server/Localhost. It should allow you to join your new Minecraft world.


6. Port Forwarding

If you want to allow friends or other players to connect to your Minecraft server, you will need to set up port forwarding in your router. This involves configuring your router to redirect external traffic to your Docker server's IP address and the port you specified. This way, other players can join your server using your external IP address and the chosen port.


7. Running Multiple Minecraft Servers

One of the advantages of using Docker is the ability to run multiple Minecraft servers on the same machine. To do this, use the run command again, but now, change the port (example -p 25566:25565), server name(--name mc2), and data directory for each new server. It enables you to manage and play on multiple Minecraft worlds simultaneously.


Conclusion

In this tutorial, we've shown you how to install and set up Minecraft servers using Docker, with the bonus of running multiple servers on the same machine. Docker simplifies server management, and with a few configurations, you can create an enjoyable Minecraft gaming experience for you and your friends.
